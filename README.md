## Publicación rápida en Netlify
1. Cambia el teléfono, correo, horarios y áreas de servicio en `index.html`.
2. Sustituye las reseñas de ejemplo y las fotografías temporales.
3. Confirma antes de publicar cualquier afirmación como “Licensed & Insured”.
4. Arrastra la carpeta completa o el archivo ZIP al panel de Netlify.
5. El formulario está preparado con Netlify Forms.
